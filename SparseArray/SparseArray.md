# SparseArray in C++

## Fucntions
1. initial_2d_array : initialize a 2d array

2. print_2d_array : print a 2d array

3. to_sparse_array : change a normal 2d array to a 2d sparse array

4. to_normal_array : change a 2d sparse array to a normal array
